# messager
## 协议格式说明 
#### 自定义协议
```
/*
	版本: 			1.0.0
	id长度:			4位
	内容长度:		2位,Content(0~65525位)
	序列化:			json
	格式:			id(4)+len(2)+content(0~65535)
*/

/*
	版本: 			2.0.0
	序列化:			json
	格式:			len(2) + content(0~65535)
*/

/*
	版本:			3.0.0
	id长度:			4位
	内容长度:		2位, content(0~65535)
	序列化:			protobuf
	格式:			id(4)+len(2)+content(0~65535)
*/

/*
	版本:			"4.0.0"
	序列化:			json
	时间戳:			当前时间到起始时间的毫秒数
	格式:			消息总长(2)+操作码(4)+发送者(8)+收信人名单长度(2)+收信人名单(0~65535)+时间戳(8)+内容(0~65535)
*/
```

#### rtsp协议
```
/*
	版本:			"rtsp"
	格式:			rtsp://[用户名]:[密码]@[ip]:[port]/[编码(h264,mpeg4...)]/[通道号]/[码流类型，主码流main,辅码流sub]/av_stream
 */
```
